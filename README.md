# AI Workflow Simulations — Locked Preview

A public, locked preview of an internal "AI Workflow Simulations" catalogue. It
shows the breadth of the portfolio (number of use cases, business domains,
deployment posture, criticality) while keeping every use case gated.

The full interactive simulations (titles, descriptions, tools, workflows and
adversarial scenarios) are intentionally **not** part of this repository or the
shipped bundle, so there is nothing sensitive to unlock client-side.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build   # outputs static files to ./dist
```

Deployed as a static site on Render (see `render.yaml`).
